Webplayer Hotkeys

Getting Started

To start using the extension, you need to set the hotkeys for it. You can do it by going to the extensions configuration page (Click more > More tools > Extensions) and, on the bottom of the list, click Keyboard Shortcuts.
On the list that opens up, find the Webplayer Hotkeys section, and there you can assign wherever hotkeys you want for play, pause and next/previous song.
